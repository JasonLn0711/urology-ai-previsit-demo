# V1 Priority Flow Shortlist

Date: 2026-04-23
Status: proposed default until 許醫師 confirms or changes it

## Purpose

Phase 0 should not force reviewers to evaluate all 12 complaint categories at equal depth. The first review should use a provisional shortlist, then ask 許醫師 to confirm, replace, or reorder it.

## Proposed First Three Flows

| Priority | Flow | Why this should be reviewed first | Current v1 evidence fit |
| ---: | --- | --- | --- |
| 1 | 頻尿或夜尿 | Common outpatient problem; tests storage/nocturia summary, diary-language boundary, and whether symptom-score reminders stay confirmation-only. | Maps to synthetic frequency/nocturia case and TUA OAB/nocturia concepts. |
| 2 | 小便困難或尿不出來 | Tests voiding/retention wording, nurse repair urgency boundary, uroflow/PVR/IPSS confirmation language, and no-triage/no-order discipline. | Maps to synthetic emptying-difficulty case and LUTS/BPO concepts. |
| 3 | 血尿或健檢發現潛血 | Tests hematuria wording, avoids cancer inference, and checks whether urine/blood/ultrasound/X-ray reminders remain physician-confirmed. | Supported by QA matrix and a dedicated synthetic visible-blood / health-check occult-blood case. |

## Why Not Start With All 12

All 12 categories are useful source material, but reviewing all of them first creates three risks:

- reviewers may give broad approval without line-level wording evidence
- exam-prep reminders may drift into ordering language
- v1 may look like a production clinical pathway instead of a synthetic product preview

Three flows are enough to test the system pattern:

- one storage/nocturia flow
- one voiding/retention flow
- one hematuria/health-check flow

If these pass, the same review method can expand to the remaining nine categories.

| 突發腰痛 | Clinic wants flank-pain/exam-prep readiness first | Tests pain wording and imaging/X-ray confirmation language, but has higher triage-risk sensitivity. |

## Review Questions Per Flow

For each selected flow, answer:

1. What should the patient/family be asked before the physician enters?
2. What should nurse/staff repair or confirm?
3. What should appear in the physician summary first?
4. Which exam-prep reminders are acceptable before physician confirmation?
5. Which words sound like diagnosis, triage, treatment, or automatic exam ordering?
6. What must remain hidden until nurse/physician review?
7. Should this flow continue, revise, narrow, pause, or go to governance review?

## Decision Rule

The shortlist is accepted only if 許醫師 explicitly confirms it or revises it.

Until then, treat it as a planning default for the Phase 0 session script, not as a clinical priority ranking.
