Patient Nurse Clinician Visit Packet Reviewer

Reviewer entrypoint

Evidence review packet

Judge whether the workflow should continue, be revised, or stop based on synthetic case evidence, not demo polish.

Synthetic data only. No diagnosis, triage, or treatment advice. Physician review remains required.

Review sequence

Run in order

  1. Open the patient intake.
  2. Load a synthetic case.
  3. Inspect nurse missing-field repair.
  4. Inspect clinician summary and visit packet.
  5. Run `npm run experiment:phase1`.
  6. Capture reviewer feedback and complete the scorecard.

Stop immediately

Revise before continuing if any screen implies diagnosis, triage, treatment, real patient-data use, or hidden missing information.

Decision frame

Evidence supports another prototype cycle within the same safety boundary.
Revise

Stop
